# Implementation Plan: CLI Music Player

**Branch**: `001-cli-music-player` | **Date**: 2026-03-17 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/001-cli-music-player/spec.md`

## Summary

Build a terminal-based music player in Go that scans a directory for
MP3 and OGG files, displays track metadata in an interactive table
(Bubble Tea + bubble-table), and plays audio via gopxl/beep. Supports
search, artist/album/genre grouping, and standard playback controls.

## Technical Context

**Language/Version**: Go 1.22+
**Primary Dependencies**: charmbracelet/bubbletea, charmbracelet/lipgloss, evertras/bubble-table, gopxl/beep v2, dhowden/tag
**Storage**: N/A (in-memory only; reads files from disk)
**Testing**: `go test` with standard testing package
**Target Platform**: Linux (ALSA required; PulseAudio/PipeWire via ALSA compat)
**Project Type**: CLI / TUI application
**Performance Goals**: Library of 1000 tracks loads in <3s; search filters in <100ms
**Constraints**: Single binary, zero runtime config, CGo required (ALSA)
**Scale/Scope**: Local music libraries up to ~10,000 tracks

packages are internal to this application.
